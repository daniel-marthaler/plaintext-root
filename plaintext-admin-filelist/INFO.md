# plaintext-admin-filelist

## Purpose
File system browser and management module providing administrative access to view and manage files in the application.

## Key Features
- File system browsing
- Directory navigation
- File listing
- File properties viewing
- File search capabilities

## Main Components
- File browser UI
- Directory tree navigation
- File management services
- File list menus

## Dependencies
### External Dependencies
- Spring Boot
- PrimeFaces
- Apache Commons IO

### Internal Module Dependencies
- plaintext-root-common
- plaintext-root-menu

## Configuration
- Root directory settings
- Access restrictions
- Allowed file types
