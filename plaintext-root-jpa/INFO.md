# plaintext-root-jpa

## Purpose
Provides JPA/Hibernate configuration and base entity classes for database persistence across the Plaintext application.

## Key Features
- JPA entity base classes
- Hibernate configuration
- Database schema management
- Audit trail support
- Custom JPA converters

## Main Components
- Base entity classes
- JPA repository configurations
- Hibernate settings
- Database migration support

## Dependencies
### External Dependencies
- Spring Boot Data JPA
- Hibernate
- H2 (PostgreSQL mode)

### Internal Module Dependencies
- plaintext-root-interfaces
- plaintext-root-common

## Configuration
- JPA configuration in application.properties
- Hibernate dialect settings
- Entity scanning packages
