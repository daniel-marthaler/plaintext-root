# plaintext-admin-sessions

## Purpose
Session management and monitoring module for viewing active user sessions, session details, and forced session termination.

## Key Features
- Active session monitoring
- Session details viewing
- User session tracking
- Session termination
- Session statistics
- Session timeout configuration

## Main Components
- Session repositories
- Session monitoring services
- Session management UI
- Session menus

## Dependencies
### External Dependencies
- Spring Boot
- Spring Security
- PrimeFaces

### Internal Module Dependencies
- plaintext-root-common
- plaintext-root-menu

## Configuration
- Session timeout settings
- Maximum concurrent sessions
- Session tracking options
