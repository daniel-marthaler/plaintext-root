# plaintext-admin-wertelisten

## Purpose
Value list (dropdown/lookup) management module providing centralized management of reference data used throughout the application.

## Key Features
- Value list definitions
- Dropdown option management
- Code/value pairs
- Value list categories
- Sorting and ordering
- Active/inactive status

## Main Components
- Value list repositories
- Value list management services
- Value list admin UI
- Value list menus

## Dependencies
### External Dependencies
- Spring Boot Data JPA
- PrimeFaces

### Internal Module Dependencies
- plaintext-root-common
- plaintext-root-menu

## Configuration
- Value list categories
- Default values
- Sorting preferences
